## 🎬 Technical Specifications

**Video Format:**
- Codec: H.264 (MP4)
- Resolution: 1920×1080 (minimum)
- Frame Rate: 60 FPS
- Bitrate: 8-15 Mbps (high quality)
- File Size: <500MB

**Audio Format:**
- Format: AAC Stereo
- Sample Rate: 44.1 kHz or 48 kHz
- Bitrate: 320 kbps (high quality)
- Voice-over: -6dB to -3dB
- Background music: -18dB to -15dB
- Sound effects: -9dB to -6dB

## 💡 Key Tips

### Do's ✅
- Show real data and real features
- Keep pace fast (no dead time)
- Emphasize business value
- Use smooth animations
- Include specific numbers (247, ₹2.4M, 87-92%)
- Show complete workflows
- Demonstrate AI autonomy
- End with clear call to action

### Don'ts ❌
- Don't use placeholder data
- Don't include loading screens or waiting
- Don't talk while recording screen (add voice-over later)
- Don't show errors or mistakes
- Don't use generic corporate music
- Don't have jump cuts or abrupt transitions
- Don't emphasize technical details
- Don't make it longer than 5:00 minutes
