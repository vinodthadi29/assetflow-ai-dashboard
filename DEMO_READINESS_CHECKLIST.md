## RECORDING EXECUTION

### Pre-Recording Checklist
- [ ] All applications closed except browser + recording software
- [ ] Browser cache cleared
- [ ] Demo data loaded and application refreshed
- [ ] Recording software levels tested
- [ ] Microphone levels tested
- [ ] Test recording done (check quality)
- [ ] Recording resolution confirmed (1920×1080)
- [ ] Frame rate confirmed (60 FPS)
- [ ] Audio sync tested
- [ ] First segment ready to record

### Recording Strategy
- [ ] Plan to record 12 segments separately
- [ ] Record each segment 2-3 times (get multiple takes)
- [ ] Pause 2-3 seconds between actions for editing points
- [ ] Move mouse deliberately (slower than normal)
- [ ] Click clearly and precisely
- [ ] Don't talk while recording (voice-over added later)
- [ ] Create a script with segment timing
- [ ] Take breaks every 3-4 segments
- [ ] Check audio levels frequently
- [ ] Save files with clear naming (Segment_1_Take1, etc.)

### Segment Recording Order (Suggested)
1. Opening slide/logo animation (standalone)
2. Login flow (smooth, deliberate typing)
3. Dashboard overview (pan and zoom)
4. Asset management (CRUD operations)
5. Allocation workflow (conflict demo)
6. Booking calendar (overlap prevention)
7. Maintenance system (ticketing)
8. Audit verification (QR scanning)
9. AI Copilot (command execution)
10. Command Palette (quick navigation)
11. Analytics dashboard (chart animations)
12. Closing slide/call to action

## VOICE-OVER RECORDING

### Voice-Over Setup
- [ ] Professional microphone connected
- [ ] Audio recording software ready (Audacity, Adobe)
- [ ] Quiet room selected
- [ ] Script printed or on second monitor
- [ ] Water nearby (for hydration)
- [ ] Audio levels tested
- [ ] Test recording done
- [ ] Background noise minimized

### Voice-Over Recording
- [ ] Record each segment voice-over separately
- [ ] Speak clearly and confidently
- [ ] Avoid filler words ("um", "uh", "like")
- [ ] Maintain professional, enterprise tone
- [ ] Speak at conversational pace (not rushed)
- [ ] Emphasize business value over features
- [ ] Record 2 takes for each segment (keep best)
- [ ] Save with clear naming (VO_Segment_1, VO_Segment_2, etc.)
- [ ] Export in lossless format (WAV or AIFF)
- [ ] Total voice-over duration: 4:00-5:00 minutes

### Voice-Over Tips
- Focus on what the feature DOES (outcome), not HOW it works (mechanism)
- Use active voice ("conflicts are prevented" → "prevents conflicts")
- Include specific numbers (247 assets, ₹2.4M, 87-92%, etc.)
- Pause for key moments to let visuals sink in
- Maintain consistent tone throughout
- Avoid technical jargon (enterprise jargon OK, coding jargon not OK)
- Sound confident, not uncertain

## CONTINGENCY PLANS

### If Recording Issues Occur
- [ ] Recording stutters → Reduce resolution to 1920x1080, close extra apps
- [ ] Audio issues → Test microphone again, restart recording software
- [ ] Sync problems → Re-sync voice-over track carefully
- [ ] Timeline issues → Re-edit and verify timing carefully
- [ ] Quality concerns → Re-record specific segment

### If Editing Issues Occur
- [ ] Crashes → Save frequently, keep backups
- [ ] Memory issues → Close other applications, reduce resolution
- [ ] Export problems → Try different export settings, try different codec
- [ ] Color issues → Recalibrate monitor, adjust color grading

### If Time Issues Occur
- [ ] Video too long → Cut non-essential features, tighten voice-over
- [ ] Video too short → Add more detail, include additional workflows
- [ ] Recording takes too long → Focus on best takes, skip perfecting every detail
